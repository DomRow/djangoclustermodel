Welcome to ClusterAt

Algorithm

The k-means clustering algorithm processes raw data and forms groups called clusters. These are calculated based on a set of attributes from the data. This allows information to be derived from the resulting groups in the form of classes.

Details

First select a dataset. A dataset is a collection of items that share common attributes. This data is what you will be exploring.

Second, select the algorithm you want to use.

Then choose the number of clusters or groups.

Finally, choose the number of items you want clustered.
