First select a dataset. A dataset is a collection of items that share common attributes. This data is what you will be exploring.

Second, select the algorithm you want to use.

Thirdly, choose the number of clusters or groups.

Finally, choose the number of items you want clustered.

Algorithm

Clustering algorithms take raw data and group them based on a set of attributes. Once this is done, information can be derived from the resulting groups. As long as these groups are robust to algorithm iterations, they're legitimate.
